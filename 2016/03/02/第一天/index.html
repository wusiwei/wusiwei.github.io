第一天

使用hexo的第一天
搭建hexo
下载node.js 自动安装即可
安装Git 去github下载msysgit最新安装文件 安装即可
安装hexo
利用npm命令安装即可 npm install -g hexo
创建hexo文件夹 再创建的文件夹下
使用命令 hexo init
安装依赖包 npm install
本地查看 执行命令hexo generate ; hexo server 在本地浏览器里查看local:4000;
注册Github 账号
创建repository 名字为wusiwei.github.io
注意在Github settings SSH keys中上传ssh秘钥
先查看是否有秘钥 cd ~/.ssh
生成秘钥 $ ssh-keygen -t rsa -C “wusiwei1122@163.com” (填的邮箱)
按3个回车，密码为空这里一般不使用密钥。
xclip -selection c id_rsa.pub 用此命令复制秘钥
将秘钥添加到github上。
测试一下 ssh git@github.com 完成
部署
编辑_config.yml(在H:\hexo下)。你在部署时，要把下面的wusiwei都换成你的账号名。
deploy:
type: git
repository: https://github.com/wusiwei/wusiwei.github.io.git
branch: master
(注意hexo3的type是git不是github且要运行命令 npm install hexo-deployer-git –save 重新部署一下)
完成

编写博客在 hexo\source_posts下（一些个性化设置百度就行）
